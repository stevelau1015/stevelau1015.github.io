机器学习：手推SVM超平面分界线

2021-04-15 9 min read # Machine Learning

问题:

训练集有3个数据点, 其中两个正例点,一个负例点: x1=(−1,2)T,x2=(1,2)T,x3=(−1,−2)Tx_{1}=(-1,2)^{T}, x_{2}=(1,2)^{T}, x_{3}=(-1,-2)^{T}x1​=(−1,2)T,x2​=(1,2)T,x3​=(−1,−2)T，试求最大间隔分离超平面 (在二维空间即为一条直线)。

模型:

设超平面方程为w1x+w2y+b=0w_{1}x+w_{2}y+b=0w1​x+w2​y+b=0，则上述问题等价于下列约束最优化问题:

min⁡wi,b,i=1,212(w12+w22) s.t. −w1+2w2+b≥1w1+2w2+b≥1w1+2w2−b≥1\begin{aligned} &\min _{w_{i}, b, i=1,2} \frac{1}{2}\left(w_{1}^{2}+w_{2}^{2}\right)\\ &\text { s.t. }\\ &-w_{1}+2 w_{2}+b\geq1\\ & w_{1}+2 w_{2}+b\geq1\\ & w_{1}+2 w_{2}-b\geq1 \end{aligned} ​wi​,b,i=1,2min​21​(w12​+w22​) s.t. −w1​+2w2​+b≥1w1​+2w2​+b≥1w1​+2w2​−b≥1​

代数解法

构建拉格朗日函数

L(w1,w2,b,μ1,μ2,μ3)=12(w12+w22)+μ1(1+w1−2w2−b)+μ2(1−w1−2w2−b)+μ3(1−w1−2w2+b)\mathcal{L}\left(w_{1}, w_{2}, b, \mu_{1}, \mu_{2}, \mu_{3}\right)=\\ \frac{1}{2}\left(w_{1}^{2}+w_{2}^{2}\right)+\mu_{1}\left(1+ w_{1}-2 w_{2}-b\right)+\mu_{2}\left(1- w_{1}-2 w_{2}-b\right)+\mu_{3}\left(1- w_{1}-2w_{2}+b\right)\\ L(w1​,w2​,b,μ1​,μ2​,μ3​)=21​(w12​+w22​)+μ1​(1+w1​−2w2​−b)+μ2​(1−w1​−2w2​−b)+μ3​(1−w1​−2w2​+b)

求偏微分

∂L1∂w1=w1+μ1−μ2−μ3=0∂L2∂w2=w2−2μ1−2μ2+μ3=0∂L3∂b=−μ1−μ2+μ3=0\frac{\partial \mathcal{L}_{1}}{\partial w_{1}}=w_{1}+ \mu_{1}- \mu_{2}-\mu_{3}=0\\\\ \frac{\partial \mathcal{L}_{2}}{\partial w_{2}}=w_{2}-2 \mu_{1}-2 \mu_{2}+\mu_{3}=0\\\\ \frac{\partial \mathcal{L}_{3}}{\partial b}=-\mu_{1}-\mu_{2}+\mu_{3}=0\\\\ ∂w1​∂L1​​=w1​+μ1​−μ2​−μ3​=0∂w2​∂L2​​=w2​−2μ1​−2μ2​+μ3​=0∂b∂L3​​=−μ1​−μ2​+μ3​=0

用KKT条件对其进行约束

−w1+2w2+b−1≥0w1+2w2+b−1≥0w1+2w2−b−1≥0μi≥0,i=1,2,3μ1(−w1+2w2+b−1)=0μ2(w1+2w2+b−1)=0μ3(w1+2w2−b−1)=0-w_{1}+2 w_{2}+b-1\geq0\\ w_{1}+2 w_{2}+b-1\geq0\\ w_{1}+2 w_{2}-b-1\geq0\\ \mu_{i}\geq0, i=1,2,3\\ \mu_{1}\left(-w_{1}+2 w_{2}+b-1\right)=0\\ \mu_{2}\left(w_{1}+2 w_{2}+b-1\right)=0\\ \mu_{3}\left(w_{1}+2 w_{2}-b-1\right)=0\\ −w1​+2w2​+b−1≥0w1​+2w2​+b−1≥0w1​+2w2​−b−1≥0μi​≥0,i=1,2,3μ1​(−w1​+2w2​+b−1)=0μ2​(w1​+2w2​+b−1)=0μ3​(w1​+2w2​−b−1)=0

分情况进行讨论

2)假设μ2=0,可以计算得w1=0,w2=μ1>0,代入三个不等式得:b≥1−2μ1,b≤2μ1−1,b=−b=2μ1−1.此时μ2=0,μ1=μ3=12,w2=12,b=0。2) 假设 \mu_{2}=0, 可以计算得 w_{1}=0,w_{2}= \mu_{1}>0, 代入三个不等式得: b≥1-2 \mu_{1}, b≤2 \mu_{1}-1, b=-b=2\mu_{1}-1.\\此时 \mu_{2}=0, \mu_{1}=\mu_{3}=\frac{1}{2}, w_{2}=\frac{1}{2}, b=0。 2)假设μ2​=0,可以计算得w1​=0,w2​=μ1​>0,代入三个不等式得:b≥1−2μ1​,b≤2μ1​−1,b=−b=2μ1​−1.此时μ2​=0,μ1​=μ3​=21​,w2​=21​,b=0。

3)假设μ3=0,由前面的条件易知w1=w2=0,b−1≥0,−b−1≥0,b≥1且b<=−1，出现了矛盾。所以μ3≥0。3) 假设 \mu_{3}=0, 由前面的条件易知 w_{1}=w_{2}=0, b-1≥0,-b-1≥0,b≥1 且 b<=-1，出现了矛盾。所以\mu_{3}\geq0。\\ 3)假设μ3​=0,由前面的条件易知w1​=w2​=0,b−1≥0,−b−1≥0,b≥1且b<=−1，出现了矛盾。所以μ3​≥0。

Python实现

对上述例子用Python语言在Notebook里实验运行一下，用训练好的边界判断点[0,4],得到该点为正例。

from sklearn import svm       
X = [[-1,2],[1,2],[-1,-2]]         #X：数据特征向量,包含3个示例,用列表（list）表示
y = [1,1,0]                     #y：数据对应的分类标记,Python中常用0,1定义两类问题
clf = svm.SVC(kernel='linear') 
#clf:创建的分类器对象,导入svm的SVC类（支持向量分类）采用线性的核函数
clf.fit(X,y)                   #用训练数据拟合分类器模型

print(clf)                     #打印分类器的内容
print(clf.support_vectors_)    #打印支持向量
print(clf.support_)            #打印支持向量的下标/索引
print(clf.n_support_)          #打印对于每一个类中支持向量的个数
print(clf.predict([[0,4]]))
